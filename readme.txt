# 🎮 JUEGOS DAVID

A fun and simple HTML5 website created to share the game **Shotgun Cop Man**, styled with inspiration from the viral "Tung Tung Tung Sahur" meme.

## 🚀 Features

- Fully responsive HTML5 layout.
- Meme-style background featuring "Tung Tung Tung Sahur".
- Eye-catching download button for the game.
- Dark theme with glowing retro-style text.

## 📁 Included Files

- `index.html`: The main web page.
- `README.md`: This documentation file.
- *(Optional)* `tung-tung-sahur.mp3`: Background music (if you decide to add it).
- *(Optional)* Local image for background if not using an online URL.

## 📥 How to Use

1. Download or clone this repository.
2. Open the `index.html` file in your browser.
3. Click the button to download the game from MediaFire.
